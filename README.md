# Hello, It's Me

## Instructions

Create a file `hello-me.py`. When you run this file, you should prompt the user to enter their name, and then say hello to that user.

## Sample Checkpoints

### Sample Input 1

```
John
```

### Sample Output 1

```
Hello, John!
```

### Sample Input 2

```
Star
```

### Sample Output 2

```
Hello, Star!
```

### Sample Input 3

```
Lin
```

### Sample Output 3

```
Hello, Lin!
```
